Fonts

In the fonts folder:

  • Bootstrap Glyphicons font files (referred to from the CSS)
  • SS Gizmo icon font: ss-gizmo.css (icon reference)

The styles assume that Roboto (weights 300, 400, 400 italic, 500) and its condensed version (weights 300, 400, 700) are loaded. The present page uses the following declarations:

<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Roboto+Condensed:300,400,700" type="text/css">
<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Roboto:300,400,500,400italic" type="text/css">
