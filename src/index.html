Usage

Install the style package

npm install git+https://github.com/healthdatasafe/style-package.git

Import in your CSS

@import "tailwindcss";
@import "hds-style/css/theme.css";
@import "hds-style/css/palettes.css";

Apply a palette

<html class="palette-doctor">  <!-- or palette-patient, palette-dark -->

Reference images from CDN

<link rel="icon" href="https://style.datasafe.dev/images/logos/favicon/favicon.ico">
<img src="https://style.datasafe.dev/images/icons/badge-check.svg">
